## Getting Started

1. Clone the repository
2. Open the project in Godot Game Engine
3. Check `game/README.md` for contribution guidelines
4. Create your work in your personal folder under `people/`

---

## Development Workflow

- **Maintainers**: Only designated team leads should edit files in `game/` directly
- **Contributors**: Work in your personal folder under `people/[name]/`
- **Integration**: Changes are reviewed and merged into `game/` by maintainers
